NEBULA VENT BLOCK TEXTURES
===========================

This block requires 5 texture files (16x16 pixels each):

1. nebula_vent_base.png
   - Base block top/bottom texture
   - Suggested color: Deep purple/blue (RGB: 100, 50, 150)

2. nebula_vent_base_side.png
   - Base block side texture
   - Suggested color: Slightly lighter purple/blue (RGB: 120, 60, 170)

3. nebula_vent_tube.png
   - Tube side texture (fleshy/organic look)
   - Suggested color: Pinkish-purple (RGB: 150, 80, 200)

4. nebula_vent_tube_top.png
   - Tube orifice opening texture
   - Suggested color: Bright pink/purple (RGB: 180, 100, 220)

You can create these as simple colored squares or add more detail for an organic/fleshy appearance.
The tube should look like a worm-like mouth opening.
